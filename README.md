# homebrew-cmdx

Community-maintained Homebrew tap for [`cmdX`](https://github.com/YONN2222/cmdX), created by [`YONN2222`](https://github.com/YONN2222). A macOS app that adds Cmd+X cut functionality to Finder.

## Installation

First, add the tap:

```bash
brew tap thedavidwenk/homebrew-cmdx
```

Then install cmdX:

```bash
brew install --cask cmdx
```

## Notes
  •	This tap is maintained by [`thedavidwenk`](https://github.com/thedavidwenk) with the author’s permission.
  
  • macOS may show a Gatekeeper warning because the app is not notarized. You can allow it via:
System Settings → Privacy & Security → Security → “Open Anyway”.

  •	cmdX may require Accessibility permissions under System Settings → Privacy & Security → Accessibility.
